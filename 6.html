Now, we want each slide (section tag) to be the full size of the viewport, so we’ll set the width of each slide to 100vw which means “100 percent of the viewport width”, and 100vh, “100% of the viewport height.” 

    section {  width: 100vw; height: 100vh; }
Of course, that’s going to mean that only one slide will fit on the screen, as you can see in the next slide.
