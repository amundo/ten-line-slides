Okay, we can listen to arrow keys now.

But what should we do when the user hits a key?

Finally we’re in a postion to show the heart of this whole business.

Let’s start with advancing to the next slide. In order to advance to the next slide, we just take the first section in the main tag, and move it to the end. That’s surprisingly easy:

let main = document.querySelector('main')
main.appendChild(main.firstElementChild)

That’s it! Go to the next page and try hitting the right arrow key.
