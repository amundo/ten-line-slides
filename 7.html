Here we are in a sea of green. But try scrolling down, and you’ll see the other slides.

But slides don’t scroll, they “paginate” or something. You only see one slide at a time. Weirdly enough, the way we’re going to do that is hide all the slides, but then follow that up by showing only the first one. That looks like this: 

section { display: none; }
section:first-child { display: block; }
