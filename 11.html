Voilà. All that’s left is to handle the left arrow key. There is no .prependChild to parallel .appendChild on DOM nodes, so we’ll have to use another approach. One is to use the insertAdjacentElement method, which takes a “where” argument (see MDN for more info).

We’ll just need to hook up this function to the left arrow: 

main.insertAdjacentElement(
  'afterbegin',  // after the main tag begins…
  main.lastElementChild // …put this element
)
Now you can go to next page and try hitting the right and left arrow keys.
