And behold, scrolling does nothing, because we’re only looking at the first slide.

At this point, we face a decision. It’s pretty clear that we’re going to need some Javascript, because we have to listen to user behavior. Let’s assume that we want to use the left and right arrow keys to page through the slides. Here’s how we can do that:

document.addEventListener('keyup', keyupEvent => {
  if(keyupEvent.key == 'ArrowRight' || keyupEvent.key == 'ArrowLeft'){
    document.querySelector('section').textContent = keyupEvent.key
  }
})
